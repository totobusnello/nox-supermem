# Perfil: Assistente Pessoal

## Para que serve
Agente focado em produtividade pessoal — gerencia tarefas, acompanha agenda e mantém histórico de decisões com memória persistente entre sessões.

## Ideal para
- Freelancers e empreendedores solo
- Quem precisa de um "segundo cérebro" que não esquece nada
- Gestão de múltiplos projetos simultâneos

## Como usar
1. Copie `SOUL.md` e `HEARTBEAT.md` para a pasta do seu agente OpenClaw
2. Edite `SOUL.md`: substitua os placeholders com seu nome e contexto
3. Abra uma sessão — o agente já começa com `nox-mem primer`

## O que ele faz de diferente
- Verifica pendências abertas antes de perguntar o que você quer
- Nunca repete perguntas já respondidas (memória persistente)
- Sugere próximos passos com base no histórico

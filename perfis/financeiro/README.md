# Perfil: Analista Financeiro

## Para que serve
Agente focado em análise financeira com memória de longo prazo — acompanha KPIs, compara períodos e mantém histórico de decisões financeiras.

## Ideal para
- Controle financeiro pessoal e de negócios
- Acompanhamento de investimentos
- KPIs de startups e PMEs
- Análise de séries temporais (receita, custos, margens)

## Como usar
1. Copie `SOUL.md` e `HEARTBEAT.md` para a pasta do seu agente
2. Edite `SOUL.md`: adicione as métricas que você quer acompanhar
3. Configure o digest semanal — gera relatório automático todo domingo

## O que ele faz de diferente
- Compara métricas atuais com histórico armazenado na memória
- Alerta quando detecta variação anormal
- Mantém contexto de por que cada decisão foi tomada
- Relatório semanal automático via Ollama (digest)
